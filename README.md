## What It Is

Xios Aimbot is a compact script that alters aim behavior in supported PC games. It analyzes what is visible on screen and then applies crosshair adjustments according to the values you set. The intent is to give players more granular control over aiming response without changing the game's core files.

This build places emphasis on better handling of moving targets and on lowering input latency. Everything runs locally on the client side and does not communicate with game servers. Sensitivity, activation zones, and smoothing can all be tuned through the included configuration file.

**Can I customize the hotkeys?**  
Yes. Every input binding can be changed in the options file. See the configuration section above.

**Does this work with all games?**  
No. It is meant for titles that accept mouse input without additional protection layers. Results will differ from game to game.
